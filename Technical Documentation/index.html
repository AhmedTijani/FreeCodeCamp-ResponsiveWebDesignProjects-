Introduction

Trani was created as a platform to make online bulk transactions easier. It is powered by the Paystack API and was one of the first Full-Stack projects I did.

The aims of this document are:

  • A Quick Guide to starting with Trani
  • A General Overview
I hope I can provide a simple and clear guide for all to follow.

Getting Started

A brief explanation of the project, it is a Vue powered front end for the Paystack API. It allows you to perform the following operations:

  • Display a person's name from account
  • Transfer money in bulk to recipients
  • Save a recipient for a recurring payment
Quite simple no?

To perform a payment:

  • Step 1: Fill in the information
  • Hit the pay button
  • The recipient details will be saved for future payments

Accessing Users

User data won't be persisted to a database, instead an API call will retrieve the list of users everytime the app is loaded This way data is more secure and less prone to security risks. All we provide is a simple frontend

Some of the API calls include:

Create a new recipient

curl https://api.paystack.co/transferrecipient -H "Authorization: Bearer YOUR_SECRET_KEY" -H "Content-Type: application/json" -d '{ "type": "nuban", "name": "Zombie", "description": "Zombier", "account_number": "01000000010", "bank_code": "044", "currency": "NGN" }' -X POST

List all recipients

curl https://api.paystack.co/transferrecipient -H "Authorization: Bearer YOUR_SECRET_KEY" -X GET

List a Recipient

curl https://api.paystack.co/transferrecipient/:id_or_code -H "Authorization: Bearer YOUR_SECRET_KEY" -X GET

Update Recipient details

curl https://api.paystack.co/transferrecipient/:id_or_code-H "Authorization: Bearer YOUR_SECRET_KEY"-H "Content-Type: application/json"-d '{ name: "Rick Sanchez" }'-X PUT

Delete Recipient

curl https://api.paystack.co/transferrecipient/:id_or_code -H "Authorization: Bearer YOUR_SECRET_KEY" -X DELETE
